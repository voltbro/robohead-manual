Робоголова Bbrain 2.0 поставляется в готовом к работе виде. В базовую комплектацию входит:

## Основные компоненты

- ***Роботизированная голова в сборе***
  - Включает все предустановленные модули:
    - дисплей
    - камера
    - микрофоны
    - сервоприводы
    - динамики

- ***Подставка для настольной работы***
  - Позволяет устойчиво разместить устройство на столе. Крепится с помощью:
    - Винты `М5×12` с шестигранной головкой — 2 шт.
    - Т-гайки `M5` — 2 шт.

## Аксессуары

- **Блок питания**
  - Тип: **12В / 2.5А** с разъёмом **DC-jack**  
> 💡**Совет:** Для корректной работы используйте только оригинальный адаптер

- **Цветной шарик**
  - Для демонстрации функции отслеживания объекта (произнесите *«Слушай, Робот! Следи за шариком»*).

- **Крепление робоголовы для робота TurtleBro2**
  - Позволяет закрепить устройство на роботе TurtleBro2. Крепится с помощью:
    - Винты `М4×18` — 2 шт.
    - Гайка-барашек `M4` — 2 шт.
    - Винты `М3×16` — 2 шт.
    - Гайки `М3` — 2 шт.

## Документация

- **Инструкция по быстрому старту**
  - Содержит ключевые шаги:
    - Первое включение
    - Подключение к Wi-Fi
    - Базовые команды
